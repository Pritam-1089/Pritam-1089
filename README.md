### 🎯 Featured Project: TaskFlow

<div align="center">
  <h3>📋 Real-Time Project Management Platform</h3>
  <img src="https://img.shields.io/badge/Status-Active%20Development-00ff9d?style=flat-square&logo=check" />
  <img src="https://img.shields.io/badge/Stack-.NET%209%20%2B%20Angular-00aeff?style=flat-square" />
</div>

**TaskFlow** is a full-stack project management system built for teams that need real-time collaboration without the bloat:

✅ **Real-Time Kanban Boards** - SignalR-powered instant sync across all connected clients  
✅ **Drag-and-Drop Workflow** - Smooth task movement with optimistic UI updates  
✅ **Team Roles & Permissions** - JWT-based auth with role-based access control  
✅ **Activity Logs & Analytics** - Full audit trail and project performance dashboards  
✅ **Enterprise Grade** - Clean Architecture, SOLID principles, testable, maintainable  

**Tech Stack:** ASP.NET Core 9 | SignalR | EF Core | Angular | SQL Server | JWT
