### Featured Project

#### ShopEase - E-Commerce Platform

A full-stack e-commerce application built with modern technologies and clean architecture.

| | Repository | Tech |
|--|-----------|------|
| **Backend** | [ECommerceApp](https://github.com/Pritam-1089/ECommerceApp) | ASP.NET Core 9, EF Core, SQL Server, JWT Auth |
| **Frontend** | [ECommerceApp-Frontend](https://github.com/Pritam-1089/ECommerceApp-Frontend) | Angular 20, TypeScript, SCSS |

**Key Features:**
- JWT Authentication with Role-Based Access Control (Admin/Customer)
- Product Catalog with Categories, Search & Filtering
- Shopping Cart & Order Management
- Address Management with Pincode Auto-Fill
- Admin Dashboard for Products & Orders
- Clean Architecture (API > Application > Core > Infrastructure)
- RESTful API with Swagger Documentation
